# 💻 Aula 09: Laboratório Prático — Lista de Exercícios 04 (Arrays)

### 📝 Introdução
A nona aula foi realizada no laboratório de informática com foco total no desenvolvimento de algoritmos utilizando estruturas de dados homogêneas. O objetivo principal foi consolidar o uso prático de Vetores e Matrizes em Java.

---

### 🛠️ Atividades Desenvolvidas
Dedicamos a aula para a resolução da **Lista de Exercícios 04**, onde praticamos:
* Criação, inicialização e preenchimento dinâmico de vetores utilizando o laço `for`;
* Manipulação e busca de elementos específicos dentro de matrizes bidimensionais;
* Aplicação prática do laço `for-each` para leitura e exibição simplificada de relatórios de dados no console.
