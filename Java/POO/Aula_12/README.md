# 💻 Aula 12: Laboratório Prático — Lista de Exercícios 05 (POO Inicial)

### 📝 Introdução
A décima segunda aula foi totalmente prática e realizada no laboratório, com o objetivo de fixar os conceitos recém-aprendidos de Classes, Objetos, Atributos e Métodos através do código Java.

---

### 🛠️ Atividades Desenvolvidas
Focamos na resolução completa da **Lista de Exercícios 05**, exercitando as seguintes habilidades:
* Criação manual de classes estruturadas com diferentes tipos de atributos;
* Instanciação de múltiplos objetos independentes baseados no mesmo molde de classe;
* Escrita e chamada de métodos com e sem parâmetros para alterar ou expor o estado dos objetos criados.
