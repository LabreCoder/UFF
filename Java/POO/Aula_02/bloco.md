## O que são Blocos?

São sequências de declarações agrupadas entre chaves <b>*{}*</b>. Eles definem o escopo de variáveis e estruturam a <b>*lógica de métodos, laços (for, while) e estruturas condicionais (if, switch)*</b>. Blocos ajudam a organizar o código e podem conter instruções de controle ou inicialização.

### Exemplo

Código:

``` 
public class Calculadora { 
    // Método que soma dois números 
    public int somar(int a, int b) { 
        return a + b; 
    } 
}

Dentro do código acima, o bloco será o <b>*'return a + b;'*</b>:
