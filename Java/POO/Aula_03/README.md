# 💻 Aula 03: Configuração de Ambiente e Primeiro Laboratório

### 📝 Introdução
Na terceira aula, tivemos o nosso primeiro encontro prático no laboratório de informática. O foco inicial foi preparar as máquinas para o desenvolvimento Java e realizar os primeiros testes práticos de compilação e execução de código.

---

### ⚙️ Preparando o Ambiente (O Desafio da Instalação)
Configurar o ambiente de desenvolvimento Java pode ser um pouco desafiador no começo, pois envolve entender três componentes fundamentais:

1. **JDK (Java Development Kit):** O kit com ferramentas necessárias para compilar nossos códigos.
2. **JRE (Java Runtime Environment):** O ambiente necessário para apenas executar os programas Java.
3. **Variáveis de Ambiente:** Configuração do sistema operacional (como o `JAVA_HOME` e o `Path`) para que a máquina reconheça os comandos `javac` e `java` através do terminal.

Apesar das dificuldades iniciais comuns no processo de instalação e configuração, o ambiente foi estabelecido com sucesso, permitindo o desenvolvimento e a resolução da primeira lista de exercícios práticos da disciplina.
