# 💻 Aula 05: Laboratório Prático de Operadores e Condicionais

### 📝 Introdução
A quinta aula foi realizada no laboratório de informática com o objetivo de colocar em prática os operadores aritméticos e relacionais aprendidos anteriormente. 

---

### 🛠️ Atividades Desenvolvidas
Durante o laboratório, focamos na resolução da primeira lista de exercícios práticos, aplicando os conceitos em cenários reais de tomada de decisão utilizando as estruturas condicionais básicas do Java:
* **`if` / `else`:** Controle de fluxo para desvios condicionais baseados em testes lógicos.
* **Captura de dados:** Uso do objeto `Scanner` para ler dados digitados pelo usuário no console e processar os resultados matemáticos em tempo de execução.
