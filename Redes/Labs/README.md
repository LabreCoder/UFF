# 📌 Organização de Laboratórios de Redes - UFF

Esse repositório será destinado para os laboratórios que realizei durante a disciplina de Redes na UFF.

## 🎯 Objetivo

Esse repositório tem como objetivo demonstrar os conhecimentos que adquiri durante a realização das atividades da disciplina de Redes, onde eu descrevo as configurações que precisei realizar para poder seguir com o exercício, quanto com as respostas que tive para cada uma das questões propostas.

Com isso eu pude:

* Consolidar estudos sobre Redes;
* Documentar práticas realizadas.

## 📖 Escopo

O repositório está dividido por cada prática de laboratório proposta:

**Inclui:**

* Wireshark_HTTP
* DNS

**Não inclui ainda:**

* Socket


## 🗂️ Estrutura do Repositório

Apresente a organização dos arquivos e pastas.

```
/Labs
├── DNS
│   └── README.md
├── README.md
└── Wireshark_HTTP
    └── README.md
```

## ⚙️ Como Utilizar

Como o repostirório tem o objetivo de documentar as práticas realizadas, ele será apenas para visualização, sem executáveis. Entretanto se quiserem realizar o 'clone' do repositório, não tem problemas:

```bash
# Exemplo de instalação
git clone https://github.com/LabreCoder/UFF/tree/master/Redes/Labs
```
